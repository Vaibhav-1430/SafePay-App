# SafePay AI Modules

This folder contains modular AI/risk logic used by the Node.js backend.

## Modules
- `risk-engine.js`: transaction fraud risk scoring
- `scam-engine.js`: scam phrase detection
- `behavior-engine.js`: anomaly scoring
- `assistant-engine.js`: spending summary assistant

All modules are deterministic and free-tier friendly (no paid AI API required).
